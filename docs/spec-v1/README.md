# 棋者弈也 V1 规格包

本规格包把已经确认的产品决策整理为设计、开发、测试和部署共用的唯一基线。

## 文件

1. `01-正式产品规格.md`：产品范围、规则、角色、用户流程、验收标准。
2. `02-页面状态图.md`：全局连接、首页、房间、移动端交互及错误状态。
3. `03-服务端房间状态机.md`：服务端权威状态、数据结构、事件契约和状态转移。
4. `04-前后端拆分部署配置.md`：同一 GitHub 仓库拆成 Render Static Site 与 Render Web Service 的部署方法。
5. `render.split.yaml`：Render Blueprint 示例。
6. `.env.frontend.example`、`.env.backend.example`：本地与手工部署环境变量示例。
7. `page-state-map.svg`：页面状态图。
8. `room-state-machine.svg`：服务端房间状态机图。

## 冻结决策摘要

- 正式名称：**棋者弈也**。
- 产品说明：好友五子棋 · ONLINE GOMOKU。
- 匿名好友房，发送链接后进行一对一对弈。
- 19×19 自由五子棋，无禁手，五子及以上立即获胜，满盘判和。
- 第一局随机分配黑白；之后每局自动交换黑白。
- 前两名为棋手，之后最多 5 名观众；观众可看棋局并参与实时聊天。
- 新加入或刷新后不显示历史聊天。
- 悔棋只撤回最新一手，次数不限，但必须由对方同意；结束后不能悔棋。
- 棋手断线时暂停，不由观众补位。
- 双方明确离开后立即删除房间；异常断线不等同于离开。
- 手机端默认开启确认落子，设置保存在当前设备；支持放大镜、按钮缩放、双指缩放和拖动。
- 前端使用 Render Static Site；后端使用 Render 免费 Web Service，单实例、内存房间。
